# 데이터 센터 클라우드 (로컬)

RAW 데이터(excel · pdf · docx 등)와 Claude로 생성된 skill을 한곳에 아카이빙하는 로컬 웹 대시보드입니다.
외부 패키지 설치가 전혀 필요 없으며(파이썬 표준 라이브러리만 사용), 인터넷 없이 동작합니다.

## 실행 방법

- **간편 실행**: `start.bat` 더블클릭 → 브라우저가 자동으로 열립니다.
- **수동 실행**: 터미널에서
  ```
  python server.py
  ```
  후 브라우저에서 http://127.0.0.1:8765 접속.

종료: 터미널에서 `Ctrl + C`

## 주요 기능

- **업로드**: 드래그&드롭 또는 파일 선택(여러 개 동시 가능). 분류(RAW / Skill), 제목, 태그, 설명 입력.
- **검색/필터**: 제목·설명·태그·파일명 통합 검색, RAW/Skill 분류 탭, 정렬(최신·이름·용량).
- **미리보기 / 다운로드**: 브라우저에서 바로 열거나 내려받기.
- **편집 / 삭제**: 메타데이터 수정, 파일과 함께 삭제.
- **통계**: 전체 건수, RAW/Skill 개수, 총 용량.

## 저장 구조

```
datacenter/
├─ server.py        # 서버 (표준 라이브러리만)
├─ index.html       # 대시보드 UI
├─ start.bat        # 더블클릭 실행
├─ datacenter.db    # 메타데이터 (SQLite) — 자동 생성
└─ storage/         # 실제 업로드 파일 보관함
```

- 파일은 `storage/` 폴더에 `타임스탬프_원본파일명` 형태로 저장됩니다.
- 메타데이터(제목·분류·태그·설명·크기·SHA256 해시 등)는 `datacenter.db`에 기록됩니다.
- **백업**: `storage/` 폴더와 `datacenter.db` 파일만 복사하면 전체 아카이브가 보존됩니다.

## 설정 변경

`server.py` 상단에서 조정 가능합니다.

- `PORT = 8765` — 포트 번호
- `MAX_UPLOAD` — 업로드 최대 용량 (기본 1GB)
- `HOST = "127.0.0.1"` — 같은 네트워크의 다른 기기에서 접속하려면 `"0.0.0.0"` 으로 변경
