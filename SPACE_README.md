# Shindong Data Center

신동와인 자료 아카이빙 웹앱 (RAW 데이터 + Claude 스킬). 업로드/검색/태그/미리보기/다운로드/통계 지원.

- 접근하려면 비밀번호 로그인 필요 (`DC_PASSWORD`).
- 업로드한 데이터는 비공개 HF Dataset(`HF_REPO_ID`)에 자동 백업되어 Space 재시작 후에도 유지됨.

> 이 README는 HF Space 저장소 전용입니다(맨 위 YAML frontmatter가 Space 설정). GitHub repo 설명은 `README.md` 참고.
